一些值得阅读的开源C代码

24May

先挖坑，有空再慢慢拜读。

  1. Webbench 一个在linux下使用的网站压测工具。Web Bench Homepage

  2. Tinyhttpd 一个轻量级的Http Server Tiny HTTPd

  3. CMockery Google发布的C单元测试框架。 cmockery - A lightweight library to simplify and generalize the process of writing unit tests for C applications.

  4. cJSON C语言实现的一个JSON编解码器。cJSON

  5. Libev 一个开源的事件驱动库。Libev

  6. Memcached 一个高性能的分布式内存对象缓存系统。memcached - a distributed memory object caching system

  7. Lua The Programming Language Lua

  8. SQLite 一个开源的嵌入式关系数据库。SQLite Home Page

  9. Redis 用ANSI C 编写的开源数据结构服务器。Redis

  10. Nginx 一个高性能的 HTTP 和反向代理服务器。Nginx

  11. UNIXv6 适合学习的操作系统内核源代码。UNIXv6

  12. NETBSD 一个免费的，可移植性的 UNIX-like 操作系统。 The NetBSD Project
